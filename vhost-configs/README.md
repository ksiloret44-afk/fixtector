# Configurations Virtual Host

Ce dossier contient les fichiers de configuration Virtual Host générés automatiquement.

## Fichiers générés

- `{domain}.conf` - Fichier de configuration Apache ou Nginx
- `install-{domain}.sh` - Script d'installation automatique

## Utilisation

1. Générez votre configuration via l'interface dans **Paramètres > Virtual Host**
2. Téléchargez ou copiez le fichier de configuration
3. Exécutez le script d'installation ou suivez les instructions manuelles

## Sécurité

⚠️ **Important** : Ces fichiers peuvent contenir des informations sensibles. Ne les partagez pas publiquement.
